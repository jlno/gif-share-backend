# gif-share-backend
Api de uma aplicação web para o compartilhamento de GIFs.

## Banco de dados
```
PostgreSQL
```

## Dependências globaisds
* `brew install ffmpeg graphicsmagick`

## Configuração

```
src/config/
```
- [x] .env.dev - padrão
- [x] .env.test
- [x] .env.prod

## Iniciar
* `npm install`
* `npm start`

## Teste
* `npm run test`

## Build
* `npm run build`
* `NODE_ENV=dev node index.js`
